# Vintage Landing Page

A modern vintage-inspired landing page built using HTML, CSS, JavaScript, and GSAP animations. The design combines minimalist aesthetics with elegant typography, smooth interactions, and a luxury jewelry showcase experience.

## Preview

Vintage is a stylish landing page concept for a jewelry brand, featuring a clean layout, large typography, image-focused sections, and subtle animations that create a premium user experience.

## Features

* Elegant vintage-inspired design
* Responsive landing page layout
* Interactive navigation menu
* GSAP-powered animations
* Modern typography and visual hierarchy
* Product showcase section
* Smooth hover and transition effects
* Lightweight and fast-loading structure

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* GSAP (GreenSock Animation Platform)
* Remix Icon Library



## Components

### Navigation Bar

* Brand logo/title
* Navigation links
* Interactive menu icon

### Hero Section

* Large promotional heading
* Scroll indicator
* Visual-focused product presentation

### Product Showcase

* Jewelry collection highlights
* Image gallery layout
* Collection information and slide counter

## Animation

The project uses GSAP for creating smooth transitions and engaging visual effects. Additional JavaScript interactions include:

* Menu click animation
* Scale transformation effects
* Dynamic shadow and border-radius transitions
